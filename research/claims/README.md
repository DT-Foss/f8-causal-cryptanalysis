# Claim evidence registry

`claim_evidence.csv` is the authoritative map from manuscript claims to local
evidence. A material claim must identify:

1. manuscript and line,
2. the exact executable script,
3. retained raw data or result,
4. an exact reproduction command,
5. a status and limitation.

The registry covers both submitted manuscripts. It is intentionally stricter
than a bibliography: citations establish external context, while this registry
tracks whether this repository can reproduce the authors' own empirical claims.
